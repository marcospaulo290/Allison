ALLISON PROJECT  

15/04/2021 - Quinta Feira 
O projeto Allison é uma assistente virtual que foi inspirada no JARVIS do Iron Man.
O nome Allison veio do no meu colega chamado Allison que morava no mesmo apartamento, ele é muito trabalhador. O projeto é uma construção de assistente virtual, que, em sua essência é servidor ao seu usuário, fica a homenagem ao meu colega. 

PS: O projeto terá um assistente virtual do gênero feminino então ela se chamará Allison com o pronunciamento de Élisson. 

Detalhe importante, eu não comecei esse projeto com nenhuma finalidade além de praticar um pouco a linguagem python.
